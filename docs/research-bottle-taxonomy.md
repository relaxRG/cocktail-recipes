## 专业分区框架
六大基酒(base spirits): Gin、Vodka、Rum、Tequila/Agave、Whiskey、Brandy。
Modifiers(spirited modifiers)五类: Vermouth(强化芳香酒)、Herbal/Spiced Liqueurs、Amaro/Aperitivo、Bitters、Fruit & Infused Liqueurs。
其他: Fortified wine 家族 = Port/Sherry/Madeira/Marsala/Vermouth(aromatized)。

## 风格子分类(styles)
### Gin
London Dry / Plymouth / Old Tom / Navy Strength / Contemporary(New Western) / Flavored & Sloe(gin liqueur) / Genever(独立类,金酒祖先) / Aged Gin
### Tequila & Agave
Blanco(Silver) / Joven(Gold) / Reposado(2-11月) / Añejo(1-3年) / Extra Añejo(>3年) / Cristalino / Mezcal(Joven/Reposado/Añejo,烟熏) / Raicilla / Sotol / Bacanora
### Whiskey
Bourbon / Rye / Tennessee / Scotch Single Malt / Scotch Blended / Irish / Japanese / Canadian / American Single Malt / Corn/Wheat
### Rum
White/Light / Gold / Dark / Black / Aged(Añejo) / Spiced / Overproof / Rhum Agricole / Cachaça / Demerara / Jamaican Funk
### Vodka
Wheat / Rye / Potato / Grape / Flavored / Craft
### Brandy
Cognac(VS/VSOP/XO) / Armagnac / Calvados / Apple Brandy/Applejack / Pisco / Grappa(pomace) / Fruit Brandy(Eau-de-vie) / Spanish Brandy
### Vermouth(苦艾酒/味美思)
Dry(法式) / Sweet Red(意式) / Blanc/Bianco / Rosé / Punt e Mes 类
### Amaro & Aperitivo
Aperitivo(Aperol/Campari,轻苦开胃) / Amaro Leggero(轻) / Amaro Medio(Nonino/Montenegro) / Fernet(重苦) / Carciofo(Cynar) / Alpine / Rabarbaro
### Bitters(苦精)
Aromatic(Angostura/Peychaud's) / Orange(Regans' No.6/Angostura Orange/Fee Brothers) / Citrus(grapefruit等) / Spice/Mole(Bittermens Xocolatl) / Tiki(Elemakule) / Celery/Savory / Fruit(Cherry Bark Vanilla 等)
### Liqueurs(利口酒)
Orange/Triple Sec(Cointreau/Grand Marnier) / Herbal(Chartreuse/Benedictine/Absinthe) / Fruit / Coffee / Cream / Nut / Floral(St-Germain 等) / Anise
### Fortified Wine
Port / Sherry(Fino/Oloroso/PX) / Madeira / Marsala
### Juice & Soft Drinks
Citrus Juice(柠檬/青柠/橙/西柚) / Other Juice(菠萝/蔓越莓/番茄) / Soda Water / Tonic / Ginger Beer/Ale / Cola & Soft Drinks / Syrup(糖浆归原材料)

## 备注
- Liqueur ABV 通常 15-30%(15-60 proof),烈酒通常 ≥40%。
- Genever 通常单列不算 gin;Mezcal 与 Tequila 并列于 Agave 烈酒。
- Navy Strength ≥57% ABV;London Dry 无添加(≤0.1g/L 糖)。
