# 本轮任务:标签系统双语化(Tag Manager 中英文)
用户需求:
1. 标签(分类cat?/基酒/杯型/风味)增加独立中英文名字段,不合并在一个标题
2. 新建标签时自动键入另一语言译名(内置词典)
3. 类似酒库展示:中文界面中文优先英文为辅,英文界面反之;列表/筛选/表单/详情全局同步

关键文件:
- lib/recipes/types.ts: TagItem 定义 (line ~95+), DEFAULT_* 常量, mk()
- lib/recipes/store.tsx: tags CRUD (addTag/renameTag等), TAG 持久化
- app/(tabs)/categories.tsx: 标签管理页(分组管理卡+renderTagRow)
- app/recipe-form.tsx: 基酒/杯型/风味 chips
- app/(tabs)/index.tsx: 首页筛选 chips
- app/recipe/[id].tsx: 详情显示 baseSpirit/glass/flavors(存的是标签 name 字符串)
- lib/utils.ts: displayNames(zh, en, lang) 已有双语工具

注意:Recipe.baseSpirit/glass/flavors 存的是标签名字符串(中文),改动需保持引用兼容:
方案:TagItem 增加 nameEn 字段;name 保持中文主键不变(避免破坏 recipe 引用);
展示时按语言优先: zh界面 name(nameEn辅), en界面 nameEn(name辅)。
新建时:输入任一语言,词典自动补另一个;词典覆盖默认标签(草本Herbal/果味Fruity/柑橘Citrus/花香Floral/甜润Sweet/酸爽Sour/苦韵Bitter/辛香Spicy/烟熏Smoky/咸鲜Savory/马天尼杯Martini/古典杯Rocks/高球杯Highball/柯林杯Collins/库佩杯Coupe/飓风杯Hurricane/子弹杯Shot/金酒Gin/朗姆Rum/伏特加Vodka/威士忌Whiskey/龙舌兰Tequila/白兰地Brandy/利口酒Liqueur/无酒精Non-Alcoholic/其他Other/经典Classic/自创Original/清爽Refreshing/浓烈Strong)
分类 Category 也是独立类型(id/name/color)也需要 nameEn。
进度:刚开始,已更新 plan。
